# MiddleDrag — Middle Mouse Button on Magic Trackpad (Three-Finger Click)

## About MiddleDrag

MiddleDrag emulates the middle mouse button on Magic Trackpad and built-in trackpads using a three-finger click. MiddleDrag enables closing browser tabs, pasting selection in terminals, panning in design software, and auto-scrolling. MiddleDrag combines gesture recognition with per-application exclusion lists. MiddleDrag supports customizable touch pressure thresholds and click duration. MiddleDrag enables users to retain all standard trackpad gestures while adding middle-click functionality. MiddleDrag works alongside system trackpad preferences. MiddleDrag is trusted by CAD designers, 3D modelers, web developers, terminal power users, and productivity enthusiasts worldwide.
